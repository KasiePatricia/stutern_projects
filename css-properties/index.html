In addition to layout and format, CSS is responsible for font color and more.

CSS uses what's called the box model to tell web browsers how to display and space different elements on a web page. These boxes have four major components.
Content - The innermost part of the box. Content could be an image, text, heading, or something else.
Padding - The inner space between the content and the border of your box.
Border - The perimeter of the box. Borders can be invisible or they could be a thick colored line like the green one pictured above.
Margin - The outer space (or lack of space) surrounding the box.
The main difference between padding and margin is that padding controls how much breathing room which exists within a box(content), while margin controls how much breathing room or whitespace exists outside of a box. Padding provides the space between the border and the content of an element while Margin provides the space between the border and outer elements.
